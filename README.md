# PortfolioHUB 📂

## 🌟 Descrição

O PortfolioHUB é uma plataforma desenvolvida com o propósito de reunir, organizar e apresentar projetos digitais em um único ambiente. Este projeto foi criado como parte das atividades da disciplina de **Bootcamp** do curso de **Ciências da Computação do Centro Universitário de Brasília (CEUB)**.

O desenvolvimento e a implantação da plataforma tiveram como foco a utilização de ferramentas modernas do mercado, como Git, GitHub e Inteligência Artificial, proporcionando uma experiência prática em gerenciamento de projetos de software.

# 🚀 Objetivos da Implantação

- Criar um ambiente organizado para armazenamento de projetos.
- Aplicar conceitos de versionamento utilizando Git.
- Utilizar o GitHub para gerenciamento e compartilhamento do código.
- Implementar práticas de segurança e organização de arquivos.
- Utilizar o Google GEMINI como ferramenta de suporte durante o planejamento e desenvolvimento.

# 🔐 Segurança e Boas Práticas

Para manter o ambiente seguro e organizado, foram aplicadas as seguintes práticas:

- Controle de acesso ao repositório.
- Utilização da autenticação em dois fatores (2FA).
- Organização das versões através de commits.
- Documentação clara das alterações realizadas.
- Separação dos arquivos em pastas específicas.

---

# 🔄 Controle de Versões

O Git foi utilizado para acompanhar todas as alterações realizadas no PortfolioHUB. Cada melhoria ou atualização no projeto é registrada através de commits, permitindo rastrear o histórico do desenvolvimento e facilitar futuras modificações.

Exemplos de commits realizados:

- `Inicialização do repositório PortfolioHUB`
- `Criação da documentação principal`
- `Adição da estrutura inicial do projeto`
- `Implementação das configurações de segurança`

---

# 🧪 Testes da Implantação

Após a configuração do ambiente, foram realizados testes para verificar:

- Organização dos diretórios do projeto;
- Funcionamento do repositório no GitHub;
- Registro correto das versões através do Git;
- Aplicação das configurações de segurança.
